## Контекст
Мафия — ЧАСТЫЙ сценарий использования петлицы. Нужен отдельный UI для трекинга игроков, голосований, подозрений. ИИ анализирует речь и поведение, считает вероятности "кто мафия".

## Задача
1. Mafia Game Engine:
   - Создание игры (количество игроков, роли)
   - Фазы: день → обсуждение → голосование → ночь
   - Трекинг: кто голосовал за кого
   - Хронология всей игры в реальном времени
2. Анализ вероятностей:
   - LLM анализ речи каждого игрока
   - Логические цепочки: кто кого подозревает, кто защищает
   - Suspicion score (0-100) для каждого игрока
   - Учёт хронологии: голосования предыдущих раундов
3. Психопрофили (связка с Шелестуном):
   - Если Шелестун знает игроков (из Telegram переписок) → дополнительный контекст
   - Склонности: кто блефует, кто честен, стиль игры
4. UI:
   - Карточки игроков (имя, фото из ChromaDB, статус живой/мёртвый)
   - Вероятности (progress bar: "мафия 73%")
   - Хронология (timeline событий)
   - Подсказки владельцу (ear hints)
