# TASK_013: Подсказки на ухо + Daily Digest
**Статус:** READY_FOR_ANTIGRAVITY
**Ветка:** feature/task-013-lapel-ear-hints
**Спецификация:** docs/specs/SPEC_001_voicezettel_2_0_architecture.md — Модуль 5.4, 5.5

## Контекст
ИИ подсказывает пользователю через наушник в реальном времени — нативно, не сбивая. Аудио-роутинг: TTS → наушник, STT ← микрофон. Плюс ежедневный дайджест дня.

## Задача
1. Ear Hints Engine:
   - Анализ контекста разговора в реальном времени
   - Генерация коротких подсказок (1-2 предложения)
   - Озвучка шёпотом через TTS (низкая громкость, быстро)
   - Триггеры: пауза в разговоре, вопрос к пользователю, интересный факт
2. Аудио-роутинг:
   - Web Audio API: разделение output (TTS → наушник) и input (микрофон → STT)
   - Системный аудио-роутинг (документация для пользователя)
3. Адаптация под контекст:
   - Мафия: вероятности, подсказки по голосованию
   - Переговоры: факты о собеседнике, аргументы
   - Тет-а-тет: доп. информация, имена, даты
   - Повседневность: напоминания
4. Daily Digest:
   - Автоматическая суммаризация в конце дня (или по запросу)
   - С кем говорил, ключевые темы, решения, action items
   - Сохранение в SQLite daily_digests
   - Уведомление через Telegram бот

## Файлы для создания/изменения
- `src/modules/lapel/ear-hints.ts` — движок подсказок
- `src/modules/lapel/context-analyzer.ts` — анализ контекста для подсказок
- `src/modules/lapel/audio-routing.ts` — разделение аудио каналов
- `src/modules/lapel/daily-digest.ts` — генерация дайджестов
- `src/app/api/lapel/digest/route.ts` — API: получить/создать дайджест
- `docs/audio-routing-setup.md` — инструкция по настройке аудио

## Acceptance Criteria
- [ ] Подсказки генерируются в реальном времени (в паузах разговора)
- [ ] TTS шёпот: тихая, быстрая озвучка подсказки
- [ ] Аудио-роутинг: подсказка идёт в наушник, не слышна окружающим
- [ ] Контекстная адаптация: разные подсказки для разных сценариев
- [ ] Daily digest: суммаризация дня (спикеры, темы, решения)
- [ ] Digest сохраняется в SQLite daily_digests
- [ ] Digest можно запросить по API / голосом
- [ ] Документация по настройке аудио-роутинга
