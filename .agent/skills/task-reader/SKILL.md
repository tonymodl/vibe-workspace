name: task-reader
description: Активируется при открытии проекта и по запросу "что делать дальше", "следующая задача", "check tasks". Читает папку .tasks/ и показывает очередь задач.

# Инструкции
1. Прочитай все файлы в .tasks/
2. Отфильтруй со статусом READY_FOR_ANTIGRAVITY
3. Отсортируй по номеру (TASK_001, TASK_002...)
4. Выведи список с иконками: ✅ DONE / 🔄 IN_PROGRESS / ⏳ WAITING / 🚫 BLOCKED
5. Предложи начать с первой незавершённой задачи
