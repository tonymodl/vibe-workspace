name: smart-git
description: Активируется по "закоммить", "сохрани прогресс", "запушь", "commit", "push". Делает умные коммиты по логическим блокам.

# Инструкции
1. git status — проверь изменённые файлы
2. Сгруппируй по типу: UI отдельно, логика отдельно, тесты отдельно
3. Для каждой группы: git add [файлы] → git commit -m "тип(scope): описание"
4. git push origin [текущая ветка]
5. Если ветка новая — предложи создать PR
Типы: feat / fix / refactor / test / docs / chore
