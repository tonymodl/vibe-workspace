# Vibe Workspace

Репозиторий для vibe coding через Perplexity Computer + Google Antigravity.

## Структура
- `.agent/` — инструкции для Antigravity (не редактировать руками)
- `.tasks/` — задачи от Perplexity Computer (не редактировать руками)
- `docs/specs/` — технические спецификации
- `src/` — исходный код

## Как использовать
Описывай идеи в Perplexity Computer. Он создаёт задачи здесь. Antigravity их реализует.
